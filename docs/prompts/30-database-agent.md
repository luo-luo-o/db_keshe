# 数据库开发 Agent 提示词

你负责 Oracle 21c XE 数据库脚本、索引、序列、基础数据和查询兼容性。所有 SQL 必须能在 Oracle 21c XE 上执行。

## 关键约束

- 当前项目主键自增使用显式 `SEQUENCE`。
- 字符串使用 `VARCHAR2`。
- 数值使用 `NUMBER`。
- 采样时间使用 `TIMESTAMP`。
- 初始化脚本是重建式脚本，不做旧数据迁移。
- 采样频率固定为 1 秒一次，采样表不保存频率标记。
- 进线/出线数据使用功率因数，不设计功率类测点。
- 所有外键约束命名清晰，便于排错。

## 脚本位置

```text
Core/src/main/resources/db/oracle21c-init.sql
Core/src/main/resources/db/oracle21c-schema.sql
Core/src/main/resources/db/oracle21c-base-data.sql
Core/src/main/resources/db/oracle21c-mock-data.sql
```

`oracle21c-init.sql` 只作为入口脚本，按顺序调用 schema、base-data、mock-data。结构框架、基础数据和模拟运行数据必须分开维护。

## 核心表

### BOX_TRANSFORMER

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `TRANSFORMER_CODE VARCHAR2(64) NOT NULL`
- `NAME VARCHAR2(100) NOT NULL`
- `TRANSFORMER_TYPE VARCHAR2(50) DEFAULT 'BOX_TRANSFORMER' NOT NULL`
- `RATED_CAPACITY_KVA NUMBER(12, 3)`
- `RATED_VOLTAGE_RATIO VARCHAR2(50)`
- `COMMISSION_DATE DATE`
- `MANUFACTURER VARCHAR2(100)`
- `OIL_LEVEL NUMBER(12, 3)`
- `LOCATION VARCHAR2(200)`
- `STATUS NUMBER(2) DEFAULT 0 NOT NULL`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`
- `UPDATED_AT TIMESTAMP`

约束：

- `TRANSFORMER_TYPE` 固定为 `BOX_TRANSFORMER`。
- `TRANSFORMER_CODE` 唯一。
- `STATUS` 使用 0 正常、1 告警、2 停运。

### POWER_CIRCUIT

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `TRANSFORMER_ID NUMBER(19) NOT NULL`
- `CIRCUIT_CODE VARCHAR2(64) NOT NULL`
- `NAME VARCHAR2(100) NOT NULL`
- `DIRECTION VARCHAR2(20) NOT NULL`
- `RATED_VOLTAGE_KV NUMBER(12, 3)`
- `RATED_CURRENT_A NUMBER(12, 3)`
- `STATUS NUMBER(2) DEFAULT 0 NOT NULL`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`
- `UPDATED_AT TIMESTAMP`

`DIRECTION` 取值：

- `INCOMING`
- `OUTGOING`

### MEASURE_POINT

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `TRANSFORMER_ID NUMBER(19) NOT NULL`
- `CIRCUIT_ID NUMBER(19)`
- `POINT_CODE VARCHAR2(64) NOT NULL`
- `POINT_NAME VARCHAR2(100) NOT NULL`
- `POINT_GROUP VARCHAR2(50) NOT NULL`
- `MEASURE_TYPE VARCHAR2(50) NOT NULL`
- `PHASE_CODE VARCHAR2(20) DEFAULT 'NONE' NOT NULL`
- `UNIT VARCHAR2(20)`
- `MIN_LIMIT NUMBER(14, 4)`
- `MAX_LIMIT NUMBER(14, 4)`
- `RATE_LIMIT NUMBER(14, 4)`
- `STATUS NUMBER(2) DEFAULT 0 NOT NULL`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`
- `UPDATED_AT TIMESTAMP`

`POINT_GROUP` 取值：

- `ELECTRIC_IN`
- `ELECTRIC_OUT`
- `TRANSFORMER`
- `CABINET`

`MEASURE_TYPE` 取值：

- `VOLTAGE`
- `CURRENT`
- `POWER_FACTOR`
- `ENERGY`
- `OIL_TEMP`
- `SWITCH_STATUS`
- `FUSE_STATUS`
- `CABINET_TEMP`
- `CABINET_HUMIDITY`
- `SMOKE_STATUS`
- `DOOR_STATUS`
- `FREQUENCY`

`POWER_FACTOR` 单位可为空或使用 `PF`，阈值建议范围为 `0.0000` 到 `1.0000`。

### TS_RAW_DATA

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `TRANSFORMER_ID NUMBER(19) NOT NULL`
- `CIRCUIT_ID NUMBER(19)`
- `POINT_ID NUMBER(19) NOT NULL`
- `SAMPLE_TIME TIMESTAMP NOT NULL`
- `VAL NUMBER(14, 4) NOT NULL`
- `QUALITY_FLAG NUMBER(2) DEFAULT 0 NOT NULL`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`

`QUALITY_FLAG`：

- `0`：正常。
- `1`：可疑。
- `2`：无效。

### CABINET_DOOR_LOG

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `TRANSFORMER_ID NUMBER(19) NOT NULL`
- `POINT_ID NUMBER(19)`
- `EVENT_TYPE VARCHAR2(30) NOT NULL`
- `EVENT_TIME TIMESTAMP NOT NULL`
- `DOOR_STATUS NUMBER(1) NOT NULL`
- `NOTE VARCHAR2(500)`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`

`EVENT_TYPE` 取值：

- `OPEN`
- `CLOSE`
- `FORCED_OPEN`
- `ANTI_THEFT`

### ALARM_LOG

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `TRANSFORMER_ID NUMBER(19) NOT NULL`
- `CIRCUIT_ID NUMBER(19)`
- `POINT_ID NUMBER(19)`
- `ALARM_TYPE VARCHAR2(50) NOT NULL`
- `ALARM_LEVEL VARCHAR2(20) NOT NULL`
- `START_TIME TIMESTAMP NOT NULL`
- `END_TIME TIMESTAMP`
- `START_VAL NUMBER(14, 4)`
- `END_VAL NUMBER(14, 4)`
- `STATUS NUMBER(2) DEFAULT 0 NOT NULL`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`

### MAINT_TASK

字段建议：

- `TASK_ID NUMBER(19) PRIMARY KEY`
- `ALARM_ID NUMBER(19) NOT NULL`
- `STATUS NUMBER(2) DEFAULT 0 NOT NULL`
- `ASSIGNEE VARCHAR2(100)`
- `FEEDBACK VARCHAR2(1000)`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`
- `UPDATED_AT TIMESTAMP`
- `FINISHED_AT TIMESTAMP`

`STATUS`：

- `0`：待办。
- `1`：处理中。
- `2`：已完成。

### SYS_USER

字段建议：

- `ID NUMBER(19) PRIMARY KEY`
- `USERNAME VARCHAR2(64) NOT NULL`
- `PASSWORD_HASH VARCHAR2(255) NOT NULL`
- `DISPLAY_NAME VARCHAR2(100) NOT NULL`
- `ROLE_CODE VARCHAR2(50) NOT NULL`
- `STATUS NUMBER(2) DEFAULT 0 NOT NULL`
- `LAST_LOGIN_AT TIMESTAMP`
- `CREATED_AT TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL`
- `UPDATED_AT TIMESTAMP`

## 序列要求

每张核心表独立序列：

- `SEQ_SYS_USER`
- `SEQ_BOX_TRANSFORMER`
- `SEQ_POWER_CIRCUIT`
- `SEQ_MEASURE_POINT`
- `SEQ_TS_RAW_DATA`
- `SEQ_CABINET_DOOR_LOG`
- `SEQ_ALARM_LOG`
- `SEQ_MAINT_TASK`

## 索引要求

```sql
CREATE INDEX IDX_POWER_CIRCUIT_TRANS ON POWER_CIRCUIT (TRANSFORMER_ID);
CREATE INDEX IDX_POINT_TRANSFORMER ON MEASURE_POINT (TRANSFORMER_ID);
CREATE INDEX IDX_POINT_CIRCUIT ON MEASURE_POINT (CIRCUIT_ID);
CREATE INDEX IDX_TS_TRANSFORMER_TIME ON TS_RAW_DATA (TRANSFORMER_ID, SAMPLE_TIME);
CREATE INDEX IDX_TS_CIRCUIT_TIME ON TS_RAW_DATA (CIRCUIT_ID, SAMPLE_TIME);
CREATE INDEX IDX_TS_POINT_TIME ON TS_RAW_DATA (POINT_ID, SAMPLE_TIME);
CREATE INDEX IDX_DOOR_TRANSFORMER_TIME ON CABINET_DOOR_LOG (TRANSFORMER_ID, EVENT_TIME);
CREATE INDEX IDX_ALARM_TRANSFORMER_TIME ON ALARM_LOG (TRANSFORMER_ID, START_TIME);
CREATE INDEX IDX_ALARM_POINT_TIME ON ALARM_LOG (POINT_ID, START_TIME);
CREATE INDEX IDX_TASK_ALARM ON MAINT_TASK (ALARM_ID);
```

## 基础数据要求

`oracle21c-base-data.sql` 至少初始化：

- 1 台箱式变压器。
- 1 个进线回路。
- 至少 2 个出线回路。
- 进线/出线电压、电流、功率因数、电能测点。
- 油温、开关状态、熔断器状态测点。
- 箱式柜温度、湿度、烟雾、柜门状态测点。
- `ADMIN`、`ENGINEER`、`MANAGER` 用户。

`oracle21c-mock-data.sql` 至少初始化：

- 连续 1 秒采样数据。
- 一个功率因数或温度越限告警。
- 一条维保工单。
- 一组柜门日志。

## 查询要求

历史采样查询：

```sql
SELECT TRANSFORMER_ID, CIRCUIT_ID, POINT_ID, SAMPLE_TIME, VAL, QUALITY_FLAG
FROM TS_RAW_DATA
WHERE TRANSFORMER_ID = :transformerId
  AND SAMPLE_TIME BETWEEN :startTime AND :endTime
ORDER BY SAMPLE_TIME
```

测点曲线查询：

```sql
SELECT TRANSFORMER_ID, CIRCUIT_ID, POINT_ID, SAMPLE_TIME, VAL, QUALITY_FLAG
FROM TS_RAW_DATA
WHERE POINT_ID = :pointId
  AND SAMPLE_TIME BETWEEN :startTime AND :endTime
ORDER BY SAMPLE_TIME
```

Oracle 21c 支持 `OFFSET/FETCH`，但现阶段后端如未封装分页组件，可继续使用 `ROWNUM` 包装保持兼容风格。

## 验收标准

- 建表脚本可重复说明执行顺序。
- 表、序列、索引、外键齐全。
- 初始化数据覆盖箱变、回路、测点、账号。
- 演示数据覆盖采样、告警、工单、柜门日志。
- 查询能按箱变、回路、测点和时间范围走复合索引。
- SQL 兼容 Oracle 21c XE。
